Herukan Bot

Herukan Discord Bot ist ein geeigneter Bot für jeden Discord-Server. Er kann sowohl moderieren als auch für Unterhaltung in Ihrer Community sorgen. Der Bot wurde in der Programmiersprache Java entwickelt und basiert auf der MaddoxEngine. Der Bot wird auf unseren eigenen Zyonic Software Servern gehostet.

Herukan Dashboard

Mit Hilfe des neuen Herukan Web Dashboards ist es sehr einfach, den Herukan Discord Bot so einzurichten, dass er zu Ihrem Community Discord Server passt. Die Oberfläche des Dashboards ist einfach und benutzerfreundlich gestaltet. Derzeit wird das Dashboard noch entwickelt. Für Neuigkeiten besuchen Sie bitte die Herukan Website.

Rest Apis
